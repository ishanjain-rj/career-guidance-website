About Us

About this project

With so many endless streams in today's world, it really is impossible for one to navigate through everything on one site without getting totally confused, and that's why in this project, we have attempted to simplify all of the information in as easy way as possible to make sure that any 12th standard graduate immaterial of what field he/she is from will find the basic information of the main fields in Arts/Commerce/Science.
Engineering is one of the vast fields in Science, and it really is easy to get confused about which engineering degree you want to pursue and for that, we are having a special Aptitude Test for Engineering! All you have to do is to answer the questions in a truthful way and you'll get which course is best suited for you!

About the Creators

While creating this project, the only thing which we had in mind is to make sure that no one else goes through the same hassle we did to find a course, we empathize with the difficulty of finding a suitable college and a course and this is our small part to the society to make sure to put all the things in one place for everyone to browse. I hope it helps, sending best wishes!

~The Creating Team.
